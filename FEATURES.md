# Pick — 기능 정의서

> 실시간 강의 참여 플랫폼. 강사/스태프/학생 3가지 역할 기반.
> 최종 점검: 2026-03-24 (Playwright QA 20/20 PASS)

---

## 1. 라우트 구조

| 경로 | 역할 | 설명 |
|------|------|------|
| `/` | 학생 | 세션 참여 (QR/링크) → JoinPage → VotePage |
| `/admin` | 강사/스태프 | 로그인 → 세션 대시보드 → 수업 진행 |
| `/live` | 전자칠판 | 프로젝터용 프레젠테이션 뷰 |
| `/report` | 학생 | 개인 학습 리포트 |
| `/submit` | 학생 | 사후 과제 제출 |

## 2. 학생 기능

### 2.1 세션 참여
- **접속**: QR코드 또는 링크(`/?s={sessionId}`)로 접속
- **닉네임 입력**: JoinPage에서 닉네임 설정 (최대 10자)
- **세션 코드 표시**: 대기 화면에서 세션 코드 확인/복사 가능
- **연결 상태 배너**: 네트워크 끊김 시 "연결 끊김" 배너, 재연결 시 "다시 연결됨" 표시

### 2.2 투표/응답
- **객관식 (Choice)**: 복수 선택지 중 선택
- **O/X**: 찬반 투표
- **주관식 (Text)**: 자유 텍스트 입력
- **퀴즈 (Quiz)**: 정답 있는 객관식, 점수 + 속도 보너스
- **척도 (Scale)**: 1~5 또는 1~10 평점
- **토론 (Debate)**: 찬/반 선택 + 의견 입력
- **순위 (Ranking)**: 드래그로 순서 정렬
- **빈칸 채우기 (FillBlank)**: 빈칸에 답 입력
- **즉각적 피드백**: 투표 즉시 체크 애니메이션 + 결과 미리보기
- **타이머**: 질문별 제한 시간 (원형 링 카운트다운)

### 2.3 리액션
- 하단 리액션 바: 👍🔥❤️😂👏 (lucide 아이콘)
- 쿨다운: 3초에 1회
- 전자칠판에 버블 애니메이션으로 표시

### 2.4 손들기 / 질문
- **손들기**: 하단바에서 토글
- **긴급 질문**: 익명으로 강사에게 직접 질문
- **수업 질문 (Q&A)**: 공개 질문 + 추천 기능

### 2.5 채팅
- 실시간 공개 채팅
- 스태프 1:1 DM (도움 요청)

### 2.6 퀴즈/리더보드
- 정답 공개 화면 (맞음/틀림 + 획득 점수)
- 리더보드 (상위 5명 포디움)
- 점수: 정답 기본 100점 + 속도 보너스 최대 50점
- 업적 시스템 (연속 정답, 최고 점수 등)

### 2.7 대기 화면
- 사자 마스코트 애니메이션
- 팁 로테이션 (5초 간격)
- 참여자 수 실시간 표시
- 게임 모드 상태 표시

### 2.8 학습 리포트 (`/report?s=&p=`)
- 참여 횟수, 정답률, 점수, 순위
- 업적 배지
- 질문별 기록 (정답/오답/미응답)
- 링크 공유/복사

### 2.9 다크모드
- 학생 헤더에서 Sun/Moon 토글
- localStorage에 테마 저장 (새로고침 유지)

## 3. 강사 기능

### 3.1 로그인/회원가입
- 아이디/비밀번호 로그인
- 회원가입 (승인 대기)
- 데모 모드 (강사/스태프)

### 3.2 세션 대시보드 (탭 구조)
- **내 클래스**: 강의별 그룹, 세션 목록, 검색/필터
- **수업 기록**: 통계 개요 (총 클래스/참여자/참여율)
  - 개요: 강의별 참여율 추이 차트
  - 질문 분석: 어려웠던 질문, 최근 질문
  - **학생 참여도**: 세션별 학생 참여율/정답률/점수 테이블 + CSV 내보내기
- **질문 보관함**: 재사용 가능한 질문 라이브러리
- **과제**: 과제 관리 (§5 참조)
- **더보기**: 테마, 단축키, 프로필 편집

### 3.3 수업 진행
- **질문 관리**: 추가/수정/삭제/순서 변경 (드래그)
- **빠른 진행 카드**: 다음 질문 원클릭 활성화
- **키보드 단축키**: ← → (질문 이동), Space (다음), R (정답 공개), L (리더보드), Esc (대기)
- **참여자 관리**: 실시간 접속 목록, 투표 현황
- **QR 코드**: 학생 참여 링크 QR 표시/복사
- **내보내기 메뉴**: 질문 결과 CSV, 참여자별 응답 CSV

### 3.4 프레젠테이션 모드 (전자칠판)
- 프로젝터 최적화 (큰 글자, 높은 대비)
- 실시간 투표 결과 시각화 (바 차트, OX 배틀, 워드클라우드, Q&A 카드)
- 리액션 버블 오버레이
- 손들기/긴급 질문 알림

### 3.5 게임/이벤트
| 게임 | 설명 |
|------|------|
| 돌림판 (Roulette) | 참여자 이름 회전 → 당첨자 선정 |
| 추첨 (Lottery) | 티켓 가중 랜덤 추첨 (복수 당첨) |
| 777 슬롯 (SlotMachine) | 3릴 슬롯 → 777 당첨 |
| 핀볼 (Plinko) | 공 떨어뜨리기 → 슬롯 당첨 |
| 경품 추첨 (PrizeDraw) | 슬롯 머신 연출 → 복수 당첨 |
| 발표자 뽑기 (RandomPicker) | 이름 순환 → 감속 → 발표자 선정 |
| 쉬는시간 타이머 (BreakTimer) | 프리셋 (5/10/15/20분) + 마스코트 |
| 이해도 체크 | 5단계 이모지 선택 |
| 빠른 설문 | 1~10 숫자 평점 |
| 그룹 토론 | 주제 + 타이머 + 메모 공유 |

### 3.6 스피드 퀴즈
- 연속 퀴즈 자동 진행 (10초씩)
- 점수 + 리더보드 실시간 업데이트

### 3.7 팀 배틀
- 팀 나누기 → 팀별 점수 경쟁
- 팀 스코어보드

## 5. 과제 시스템

### 5.1 과제 관리 (강사)
- **과제 등록**: 클래스/차수 선택, 제목/설명, AI 심사 토글
- **제출 링크**: 자동 생성, 복사 버튼
- **상태 관리**: 제출 중 → 마감 → 심사 중 → 심사 완료
- **제출물 목록**: 실시간 카운트, 순위 정렬
- **CSV 내보내기**: 심사 결과 (순위/이름/평균점수/합격/심사위원별 점수)

### 5.2 과제 제출 (학생, `/submit?a={assignmentId}`)
- **랜딩 페이지**: 과제 제출하기 / 내 제출물 조회
- **제출 폼**: 이름, PIN (4자리), 프로젝트 URL, 코드 파일, PRD/기획서, 설명
- **파일 제한**: 1MB
- **수정/취소**: 이름+PIN으로 조회 → 수정/취소 가능
- **다크모드**: 토글 지원

### 5.3 AI 심사
- **7명의 AI 심사위원** (Gemini 2.0 Flash)
  - 김태호 PD, 실리콘밸리 CTO, 디자인 교수, 마케팅 CMO, UX 리서처, 보안 전문가, 스타트업 멘토
- **재시도 로직**: 30초 타임아웃, 429/503 시 2회 재시도
- **결과**: 점수(100점), 합격/불합격, 강점/약점 태그, 심사평

### 5.4 시상
- **순위 시상**: 1~3위 (대상/금상/은상)
- **특별상**: 창의력상, 디자인상, 기술력상, 실용성상
- **시상식 화면**: 프레젠테이션 모드 (ESC/X 닫기)
- **학생 결과 확인**: 이름+PIN으로 점수/합격/심사평 조회

## 6. 공통 기능

### 6.1 디자인 시스템
- **브랜드**: Indigo (#4F46E5) 악센트 + Slate 모노크롬
- **CTA**: bg-slate-900 (Linear/Toss 스타일)
- **폰트**: Pretendard (한국어) + Inter (영문/숫자)
- **마스코트**: 사자 (PickMascot)
- **다크모드**: 전체 앱 지원 (localStorage 영속)

### 6.2 PWA
- manifest.json (standalone 모드)
- Service Worker (Vite 생성)
- 설치 프롬프트 (InstallPrompt)
- 앱 아이콘 (192/512, maskable)

### 6.3 성능
- React.lazy 코드 스플리팅 (각 라우트)
- Vite 벤더 청킹 (react/firebase/motion/ui/dnd)
- Brotli + gzip 압축
- React.memo 주요 컴포넌트 (VotePage, ParticipantList, ReactionOverlay 등)
- Firebase 모듈러 import (트리 셰이킹)

### 6.4 접근성
- ARIA labels (버튼, 입력, 테이블)
- 터치 타겟 48px+
- 키보드 단축키 (관리자)
- 시맨틱 HTML

### 6.5 에러 처리
- ErrorBoundary: 라우트별 독립 에러 경계
- Firebase 리스너: 에러 콜백 포함
- 연결 상태 배너 (ConnectionBanner)
- Gemini API: 재시도 + 타임아웃

### 6.6 SEO / 소셜
- OG 메타태그 (title/description/image)
- Twitter Card 메타태그
- PWA manifest
